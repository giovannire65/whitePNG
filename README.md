# whitePNG
Questa piccola web app corregge automaticamente i pixel bianchi puri (255,255,255) presenti in un’immagine PNG, sostituendoli con (254,255,255). La differenza è invisibile a occhio nudo, ma può risolvere diversi problemi nei flussi di stampa, taglio o DTF in cui il bianco viene interpretato come trasparente.
